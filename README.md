前端基于Vue

数据库基于Mysql

后台基于koa2+sanic

+ `koa2`用于整体数据交互
+ `sanic`是一个超高效、快速的python服务端，用于执行python脚本，返回数据


- 前后端整体运行
```
npm run start
```

- 只运行node服务
```
npm run server
```
- 只运行python服务
```
python3 server/python/server.py
```

- 只运行前端UI界面
```
npm run dev
```

`tips:`
运行不成功，请检查python依赖库是否完全
